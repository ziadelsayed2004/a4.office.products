# فلو الحجز المسبق — A4

## إنشاء الحجز

1. الكاشير يفتح شاشة الحجز.
2. يضيف المنتج بالبحث أو سكان رمز المنتج.
3. المنتج يجب أن يكون قابل للحجز أو يسمح النظام بالحجز عند عدم توفر المخزون.
4. اسم العميل ورقم الموبايل إجباريان.
5. الديبوزت يحسب من نسبة المنتج أو قيمة مسموحة.
6. يتم تسجيل Payment للديبوزت.
7. يتم إنشاء Preorder مع رمز استلام آمن.
8. يتم طباعة ريسيت الحجز.
9. يزيد openPreorderQuantity للتقارير.

## استلام الحجز

1. الكاشير يسكن رمز الحجز من الريسيت.
2. السيرفر يتحقق من الرمز والصلاحية.
3. تظهر نافذة فيها بيانات العميل والمنتجات والديبوزت والمتبقي والمخزون.
4. الكاشير يحصل المتبقي.
5. النظام يتحقق من المخزون ويمنع السالب.
6. يتم خصم المخزون.
7. يتم إنشاء فاتورة/ريسيت استلام نهائي.
8. تتحول حالة الحجز إلى PICKED_UP.
9. يقل openPreorderQuantity.
