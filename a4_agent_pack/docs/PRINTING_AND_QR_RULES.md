# قواعد الطباعة والرموز — A4

## الطابعات

- طابعة ريسيت للبيع والحجز والاستلام.
- طابعة ملصقات لرموز المنتجات.

## رمز المنتج

- يشير إلى token أو SKU يتحقق منه السيرفر.
- لا يحتوي سعر المنتج ولا المخزون.
- يمكن طباعته بعدد يحدده الأدمن.
- يستخدم لإضافة المنتج للسلة بسرعة.

## رمز الحجز

- يستخدم secure token.
- يعمل فقط بعد تسجيل الدخول.
- يفتح نافذة استلام الحجز للكاشير أو الأدمن.
- لا يعرض بيانات الحجز لأي مستخدم غير مصرح.

## رمز الفاتورة

- يستخدم للرجوع الداخلي أو إعادة الطباعة.
- لا يكشف بيانات حساسة خارج النظام.

## سجل الطباعة

كل طباعة أو إعادة طباعة تنشئ AuditLog يحتوي:

- userId
- receiptId
- refType/refId
- printCount
- timestamp
- reason عند إعادة الطباعة إن وجد
