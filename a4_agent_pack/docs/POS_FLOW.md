# فلو شاشة الكاشير — A4

## البيع العادي

1. الكاشير يسجل دخول.
2. يفتح أو يستكمل شيفت نشط.
3. يضيف المنتج بسكان رمز المنتج أو البحث.
4. يختار الكمية وفئة السعر.
5. السيرفر يتحقق من المخزون.
6. يتم اختيار طريقة دفع واحدة أو أكثر.
7. يتم تنفيذ checkout.
8. يتم إنشاء Order و Payment و Receipt و InventoryLedger و AuditLog.
9. يتم خصم المخزون فوراً.
10. يتم طباعة ريسيت البيع.

## ممنوعات

- لا checkout بدون شيفت.
- لا checkout مع مخزون غير كافٍ.
- لا تعديل سعر يدوي بدون صلاحية أو سجل.
- لا إعادة طباعة بدون AuditLog.
- لا إظهار إيرادات عامة للكاشير.
