# قواعد الشيفت والتقفيل — A4

## حالات الشيفت

- OPEN
- PENDING_ADMIN_REVIEW
- CLOSED
- REJECTED

## القواعد

- لا عملية POS بدون OPEN shift.
- كاشير واحد لا يفتح أكثر من شيفت في نفس الوقت.
- الكاشير يرى شيفته الحالية وملخصها فقط.
- الأدمن يرى كل الشيفتات وكل الفروقات.
- التقفيل يقارن systemTotalsByMethod مع cashierDeclaredTotalsByMethod.
- الاعتماد أو الرفض يسجل في AuditLog.

## ملخص الكاشير

- إجمالي الكاش.
- إجمالي الكارت.
- إجمالي InstaPay والمحافظ والتحويل.
- إجمالي ديبوزت الحجوزات.
- إجمالي متبقي الحجوزات عند الاستلام.
- عدد الفواتير والحجوزات.
- المبلغ المتوقع والفروقات.
