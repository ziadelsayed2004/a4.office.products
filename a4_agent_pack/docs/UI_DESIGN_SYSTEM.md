# UI Design System

## Language

- The full app is Arabic RTL.
- Use `dir="rtl"` at root layout level.
- All visible product strings must be Arabic.
- English is allowed only for technical identifiers when unavoidable.

## Core UI areas

- Admin dashboard.
- Cashier POS.
- Products and inventory.
- Preorders and pickup dialog.
- Shift closing.
- Reports and KPIs.
- Print templates.

## Print templates

Receipts and labels must be designed for Arabic RTL thermal printing and clean QR rendering.
