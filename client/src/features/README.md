# A4 POS Features Directory

This folder contains self-contained feature slices comprising layout templates, subcomponents, api services, hooks, and localized translations for each domain area.

## Feature Slices:
- `auth/` — Session and user credentials validation UI.
- `dashboard/` — KPIs grids, operational statistics, and audit activity widgets.
- `users/` — Cashier and Administrator CRUD interfaces.
- `categories/` — Products category tags management.
- `price-tiers/` — Wholesale, student, default Retail price tier settings.
- `products/` — Catalog listings, item details form, QR labels.
- `inventory/` — Ledger transaction trails and adjustments.
- `customers/` — Direct buyer logs and contact details.
- `payments/` — Payment channels configuration panel.
- `pos/` — Cashier checkout interface, item scanning panel.
- `preorders/` — Reservation lists, deposit processing.
- `receipts/` — Invoices search and ticket reprint layout.
- `shifts/` — Shift logs, review forms, approve closures.
- `reports/` — Profit charts and spreadsheet data exports.
- `audit-logs/` — Platform logs widget.
- `printer-settings/` — Hardware size configurations.
