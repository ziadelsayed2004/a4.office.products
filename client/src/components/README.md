# Shared Design System Components

This directory houses the reusable, stateless presentation UI widgets mapped to the design system parameters:
- `navigation/` — Breadcrumbs, collapsible desktop menus, top bars.
- `feedback/` — LoadingState placeholders, confirm dialog sheets, alerts, notifications.
- `data-display/` — Dense tables, detail grids, chip states.
- `forms/` — Dense OutlinedInput text fields, dynamic select menus.
- `overlays/` — Drawer dialog sheets, modal layers.
- `print/` — Light, print-safe thermal layouts.
